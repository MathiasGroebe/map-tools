Correlation of photos to points by using the timestmap. The EXIF tags of photos are set to the postion of the point, if a matching timestamp if found.

Points with timestamp

Vector point layer with a datetime field for linking the photos to the points.

Timestamp field

Field with the timestamp of the point layer. Only field of the type timestamp are selectable.

Offset in seconds

Shift the time of the input layer for a better correlation or to fix timezone differences.

Elevation offset in meters

Apply an offset to the elevation of the input layers z coordinates.

Photo input folder

Folder with photo to be processed. Only *.jpg and *.jpeg files are supported.

Photo output folder

Folder with correlated photos and modified EXIF tags.
